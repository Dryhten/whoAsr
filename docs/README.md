### [API 上下文工程最佳实践指南](./api-context-guide.md)
基于上下文工程（Context Engineering）最佳实践的完整API使用指南，包含：

- **核心原则**：上下文完整性、音频优化、文本工程
- **实时语音识别**：WebSocket连接模式和优化策略
- **离线语音识别**：批处理优化和上下文增强
- **语音活动检测（VAD）**：上下文感知的VAD使用
- **标点恢复**：上下文增强的标点恢复策略
- **时间戳预测**：精确时间戳生成和对齐
- **高级技巧**：多模态融合、动态调整、错误恢复
- **性能优化**：模型加载、内存管理、并发控制
- **监控调试**：健康检查、性能监控、错误追踪
- **测试验证**：测试脚本和验证方法
